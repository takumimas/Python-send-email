# SendEmail-Python

# How to use

Import
```Python
from sendemail import send_email
```

Call function
```Python
send_email({body},{subject},{from_Email_add},{to_Email_add},{Email_password},{smtp_add},{port})
```
For example
```Python
send_email("test message","text","***@***.com","xxx@xxx.com","PasswordPassword","***.aaa.com","0000")
```
